## Descrição

Oprojeto criado na composição de uma tabela que conta com capitais blasileiras e a distância entre capitais. As capitais constantes nesse projeto são:

* Aracajú
* Belém
* Belo Horizonte
* Boa Vista 
* Brasília
* Campo Grande 
* Cuiabá
* Curitiba
